# Python Data Cleaner & Reporter

Beginner Python automation project: Cleans messy CSV/Excel files – removes duplicates, handles missing data, adds calculations, generates summary. Perfect demo for freelance data/Excel gigs.

Simple Python script for cleaning and processing CSV/Excel files. Removes duplicates, handles missing values, adds calculated columns, and generates a quick summary report. Built as a portfolio project to demonstrate Pandas automation skills for freelance data cleaning and Excel tasks. 

## What It Does
- Loads .csv or .xlsx
- Drops duplicates
- Fills missing values (0/mean/median)
- Adds example columns (e.g., 10% bonus on salary)
- Outputs cleaned file + console report

## Tech Stack
- Python
- pandas (data handling)
- openpyxl (Excel support)

## How to Run
1. Clone: git clone https://github.com/coderk27136/python-data-cleaner
2. Install: pip install pandas openpyxl
3. Run: python data_cleaner.py
